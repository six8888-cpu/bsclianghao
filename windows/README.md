# BSC靓号生成器 - Windows版本 💻

Windows独立可执行版本，无需Python环境。

---

## 🎯 两种使用方式

### 方式1：下载EXE（推荐）⭐

**直接下载打包好的EXE，双击运行**

#### 下载步骤：

1. **访问GitHub Actions**  
   https://github.com/six8888-cpu/bsclianghao/actions

2. **点击最新的 "Build Windows EXE"**  
   （顶部第一个，有绿色✓标记）

3. **向下滚动到 "Artifacts"**

4. **下载 `BSC-Vanity-Generator-Windows-x64.zip`**  
   （约30-50MB）

5. **解压ZIP文件**

6. **双击运行 `BSC-Vanity-Generator.exe`**

✅ **无需Python，无需安装任何依赖！**

---

### 方式2：本地运行Python脚本

如果你的Windows已安装Python 3.8+：

```bash
# 1. 克隆项目
git clone https://github.com/six8888-cpu/bsclianghao.git
cd bsclianghao\windows

# 2. 安装依赖
pip install -r ..\requirements.txt

# 3. 运行
python ultra_generator_windows.py
```

```

```

## ❓ 常见问题

### Q: 首次运行被杀毒软件拦截？
**A**: 这是正常的。PyInstaller打包的EXE会被部分杀毒软件误报。
- 添加信任/允许运行即可
- 或查看源代码自行打包

### Q: 运行后黑窗口一闪而过？
**A**: 双击运行即可，程序会自动暂停等待输入。如果还有问题：
- 右键 → "以管理员身份运行"
- 或在CMD中运行：`BSC-Vanity-Generator.exe`

### Q: 速度比Linux慢？
**A**: Windows控制台彩色输出有一定性能开销，但影响不大（<5%）。

### Q: 可以在服务器上用吗？
**A**: Windows Server可以，但推荐用Linux版本（性能更好）。

### Q: 大小写敏感吗？
**A**: 运行时会询问，可选择区分或不区分。

### Q: 可以生成多个地址吗？
**A**: 可以，运行时会询问要生成几个。

### Q: 如何停止程序？
**A**: 按 `Ctrl+C`

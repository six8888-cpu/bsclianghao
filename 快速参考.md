## 💻 常用命令

### 本地运行

```bash
# 快速启动
./start.sh

# 标准版
python3 vanity_wallet_generator.py

# 极速版（推荐超级靓号）
python3 fast_generator.py
```

### 服务器部署

```bash
# 1. 连接服务器
ssh user@server-ip

# 2. 上传文件
scp fast_generator.py requirements.txt user@server-ip:~/

# 3. 安装依赖
pip3 install -r requirements.txt

# 4. 后台运行
screen -S vanity
python3 fast_generator.py
# Ctrl+A, D 分离

# 5. 查看进度
screen -r vanity

# 6. 下载结果
scp user@server-ip:~/super_vanity_wallets.txt ./
```

## ⚙️ 推荐配置

### 进程数设置

```
进程数 = CPU核心数

示例：
8核 → 8进程
16核 → 16进程
32核 → 32进程
64核 → 64进程
```

### 服务器选择

| 目标 | CPU | 云平台推荐 | 预计成本 |
|-----|-----|-----------|---------|
| 6位 | 8核 | t3.2xlarge | $1 |
| 7位 | 16核 | c7i.4xlarge | $2 |
| 8位 | 32核 | c7i.8xlarge | $3-5 |
| 9位 | 64核 | c7i.16xlarge | $15-20 |

*使用竞价实例可节省60-90%*

## 🔧 故障排除

### 问题1：安装依赖失败

```bash
# 使用国内镜像
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题2：速度太慢

```bash
# 1. 确认进程数 = CPU核心数
nproc  # 查看核心数

# 2. 使用极速版
python3 fast_generator.py

# 3. 考虑使用更高配置服务器
```

### 问题3：SSH断开

```bash
# 必须使用screen或tmux
screen -S vanity
python3 fast_generator.py
# Ctrl+A, D 分离

# 重连后
screen -r vanity
```

### 问题4：内存不足

```bash
# 添加swap空间
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
```

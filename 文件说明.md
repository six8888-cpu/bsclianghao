## 📝 文件详细说明

### 🔥 核心程序

#### 1. fast_generator.py ⭐
**用途**：极速版生成器，支持单一匹配模式
- ✅ 前缀匹配（如：0x888888...）
- ✅ 后缀匹配（如：0x...888888）
- ✅ 包含匹配（如：0x...888888...）
- ✅ 多进程并行
- ✅ 优化版（速度显示稳定）

**适用场景**：
- 6-8位靓号
- 单一条件匹配
- 日常快速生成

**运行命令**：
```bash
python3 fast_generator.py
```

#### 2. ultra_generator.py ⭐
**用途**：超级版生成器，支持组合匹配
- ✅ 前缀+后缀同时匹配
- ✅ 支持复杂组合（如：0xc946...0bb7）
- ✅ 多进程并行
- ✅ 优化版（速度显示稳定）

**适用场景**：
- 超级靓号（8-10位）
- 前后缀组合
- 高配服务器运行

**运行命令**：
```bash
python3 ultra_generator.py
```

#### 3. ultra_generator_v2.py ✨ 新增！
**用途**：V2增强版生成器，支持组合匹配+新功能
- ✅ 前缀+后缀同时匹配
- ✅ **实时概率显示**
- ✅ **彩色界面输出**
- ✅ **运气状态提示**
- ✅ **详细统计信息**
- ✅ **峰值速度记录**
- ✅ 多进程并行

**新增功能**：
- 📊 实时显示找到概率（基于数学公式）
- 🎨 彩色输出（绿色=成功，黄色=数据，紫色=时间）
- 🎯 运气提示（😎 才刚开始 → 🔥 马上就要出了）
- 📈 生成完成后显示峰值速度和运气评估

**适用场景**：
- 超级靓号（8-10位）
- 想知道实时概率
- 喜欢美观界面
- 需要详细统计

**运行命令**：
```bash
python3 ultra_generator_v2.py
```

**推荐**：✨ 所有用户优先使用V2版本！

#### 4. ultra_generator_windows.py 🪟 Windows专用
**用途**：Windows版本，专为打包成EXE设计
- ✅ 所有V2功能
- ✅ Windows命令行优化
- ✅ 彩色输出支持
- ✅ 暂停按键支持
- ✅ 窗口标题设置
- ✅ 清屏功能
- ✅ 可打包为独立EXE

**新增功能**：
- 🪟 Windows环境检测
- 🎨 自动启用Windows彩色支持
- ⏸️ pause()暂停功能
- 🖥️ clear_screen()清屏
- 📦 mp.freeze_support()多进程支持

**适用场景**：
- Windows个人电脑
- 不想装Python环境
- 需要EXE可执行文件
- 简单到中等难度靓号（4-8位）

**打包命令**：
```bash
# Windows
build.bat

# Python
python3 build_exe.py
```

**生成文件**：
- `dist/BSC靓号生成器.exe` (约30-50MB)

### 💡 命令参数说明

**fast_generator.py 参数：**
```
模式: 1=前缀, 2=后缀, 3=包含
字符: 匹配的字符串（不含0x）
大小写: n=不区分, y=区分
数量: 需要生成几个
进程: CPU核心数（建议等于核心数）
确认: y=开始
```

**ultra_generator.py 参数：**
```
前缀: 匹配的前缀（不含0x）
后缀: 匹配的后缀
大小写: n=不区分, y=区分
数量: 需要生成几个
进程: CPU核心数（建议等于核心数）
确认: y=开始
```

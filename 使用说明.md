3. **腾讯云 CVM**

## 后台运行方法

### 使用 screen

```bash
# 创建新会话
screen -S vanity

# 运行程序
python3 vanity_wallet_generator.py

# 分离会话：按 Ctrl+A 然后按 D

# 重新连接
screen -r vanity

# 查看所有会话
screen -ls
```

### 使用 tmux

```bash
# 创建新会话
tmux new -s vanity

# 运行程序
python3 vanity_wallet_generator.py

# 分离会话：按 Ctrl+B 然后按 D

# 重新连接
tmux attach -t vanity

# 查看所有会话
tmux ls
```

### 使用 nohup

```bash
# 后台运行
nohup python3 vanity_wallet_generator.py > output.log 2>&1 &

# 查看进程
ps aux | grep vanity

# 查看输出
tail -f output.log
```

## 安全注意事项

### 私钥安全

1. **立即备份**：生成后立即多处备份
2. **离线保存**：使用U盘、硬件钱包等离线设备
3. **加密存储**：使用VeraCrypt等工具加密私钥文件
4. **纸质备份**：手写在纸上，放在保险箱

### 测试建议

1. **小额测试**：先转入少量资金测试
2. **验证私钥**：用私钥导入钱包确认能访问
3. **多次验证**：在不同钱包软件中导入验证

### 导入钱包

生成的私钥可以导入到以下钱包：

- **MetaMask**：直接导入私钥
- **Trust Wallet**：使用私钥创建钱包
- **imToken**：导入私钥功能
- **TokenPocket**：私钥导入

导入方法（以MetaMask为例）：
1. 打开MetaMask
2. 点击账户图标 → 导入账户
3. 选择"私钥"
4. 粘贴私钥（需要0x前缀）
5. 点击导入

## 常见问题

### Q1: 为什么速度比预期慢？

A: 可能原因：
- CPU性能较弱
- 进程数设置不合理
- 系统资源被其他程序占用
- 虚拟机环境性能损失

### Q2: 可以中断后继续吗？

A: 不能。每次都是随机生成，中断后需要重新开始。建议使用screen或tmux防止意外中断。

### Q3: 生成的地址安全吗？

A: 完全安全。使用Python的secrets模块生成真随机数，与标准钱包生成方法完全相同。

### Q4: 可以同时运行多个实例吗？

A: 可以。每个实例相互独立，可以设置不同的匹配模式。

### Q5: 8位以上的靓号现实吗？

A: 需要大量算力和时间：
- 9位：需要高配服务器运行1-3天
- 10位：需要运行几周到几个月
- 11位+：基本不现实，需要专业矿机级别算力

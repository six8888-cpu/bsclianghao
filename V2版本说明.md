### 5. 🧮 精确概率计算
**采用数学公式，而非简单百分比！**

**公式**：
```
P = 1 - e^(-n/N)

其中：
P = 找到概率
n = 已尝试次数
N = 总难度
e = 自然常数
```

**为什么这样计算？**
- ❌ 错误：n/N × 100% （简单百分比）
- ✅ 正确：1 - e^(-n/N) （概率论公式）

**示例**：
```
难度：1000万次
尝试：1000万次

简单计算：100%（错误！）
正确计算：63.2%（实际找到概率）
```

## 🔧 技术细节

### 概率计算原理
```python
import math

def calculate_probability(attempts, difficulty):
    """
    计算找到概率
    
    基于泊松分布近似：
    P(至少找到1个) = 1 - P(一个都没找到)
                    = 1 - (1 - 1/N)^n
                    ≈ 1 - e^(-n/N)
    """
    ratio = attempts / difficulty
    probability = 1 - math.exp(-ratio)
    return probability * 100
```

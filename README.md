# BSC链靓号钱包生成器

高性能的BSC（币安智能链）靓号钱包生成器，使用Python多进程技术，充分利用多核CPU性能。

## 🎉 V2增强版新功能

- 📊 **实时概率显示** - 知道找到的可能性，告别焦虑等待
- 🎨 **彩色界面输出** - 美观清晰的终端显示
- 🎯 **运气状态提示** - 趣味互动（😎 才刚开始 → 🔥 马上就要出了）
- 📈 **详细统计信息** - 峰值速度、运气评估
- 🪟 **Windows EXE支持** - 无需Python环境，双击即用

## 功能特点

- ⚡ **高性能**：多进程并行生成，速度可达每秒数万到数十万次（取决于CPU）
- 🎯 **灵活匹配**：支持前缀、后缀、包含、前缀+后缀组合四种匹配模式
- 🔧 **易于使用**：交互式命令行界面，简单易懂
- 📊 **实时进度**：实时显示生成进度、速度、概率
- 💾 **自动保存**：生成结果自动保存到文件

## 使用方法

运行后按照提示操作：

1. 输入前缀（如：1780，不含0x）
2. 输入后缀（如：3CffbD，无则回车）
3. 选择是否区分大小写
4. 选择需要生成的钱包数量
5. 选择使用的进程数（建议等于CPU核心数）
6. 确认开始生成

**V2版本新增显示**：
- 实时概率（0-100%）
- 运气状态（😎 才刚开始 → 🔥 马上就要出了）
- 峰值速度记录
- 运气评估

## 匹配模式说明

- **prefix（前缀）**：地址以指定字符开头
  - 例如：`888888` 会匹配 `0x888888abcdef...`
  - 难度：16^n（n为字符长度）

- **suffix（后缀）**：地址以指定字符结尾
  - 例如：`888888` 会匹配 `0x...abcdef888888`
  - 难度：16^n

- **contains（包含）**：地址包含指定字符
  - 例如：`888888` 会匹配 `0x...abc888888def...`
  - 难度：相对较低

## 难度参考

| 字符长度 | 预估尝试次数 | 预计耗时（8核CPU） |
|---------|-------------|-------------------|
| 4位     | ~65K        | < 1秒             |
| 5位     | ~1M         | ~5-10秒           |
| 6位     | ~16M        | ~1-3分钟          |
| 7位     | ~268M       | ~20-40分钟        |
| 8位     | ~4.3B       | ~6-12小时         |

*注：实际速度取决于CPU性能，上述时间基于每秒10万次的速度估算*

## 性能优化建议

1. **使用高性能CPU**：CPU核心数越多，速度越快
2. **进程数设置**：建议设置为CPU核心数
3. **云服务器**：可以使用AWS、阿里云等高配置服务器
4. **长时间运行**：对于超级靓号（8位以上），建议使用screen或tmux在后台运行

```
  地址: 0x888888a1b2c3d4e5f6a7b8c9d0e1f2a3b4c5d6e7

  耗时: 45.32 秒
```

## 技术原理

1. 生成随机32字节私钥
2. 使用椭圆曲线算法（secp256k1）生成公钥
3. 对公钥进行Keccak-256哈希
4. 取哈希值的后20字节作为地址
5. 检查地址是否符合靓号要求
6. 如不符合，重复上述步骤

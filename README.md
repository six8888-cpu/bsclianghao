# BSC链靓号钱包生成器

高性能的BSC（币安智能链）靓号钱包生成器，使用Python多进程技术，充分利用多核CPU性能。

## 功能特点

- ⚡ **高性能**：多进程并行生成，速度可达每秒数万到数十万次（取决于CPU）
- 🎯 **灵活匹配**：支持前缀、后缀、包含三种匹配模式
- 🔧 **易于使用**：交互式命令行界面，简单易懂
- 📊 **实时进度**：实时显示生成进度和速度
- 💾 **自动保存**：生成结果自动保存到文件

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python vanity_wallet_generator.py
```

然后按照提示操作：

1. 选择匹配模式（前缀/后缀/包含）
2. 输入要匹配的字符串（如：888888）
3. 选择是否区分大小写
4. 选择需要生成的钱包数量
5. 选择使用的进程数（默认为CPU核心数）

## 匹配模式说明

- **prefix（前缀）**：地址以指定字符开头
  - 例如：`888888` 会匹配 `0x888888abcdef...`
  - 难度：16^n（n为字符长度）

- **suffix（后缀）**：地址以指定字符结尾
  - 例如：`888888` 会匹配 `0x...abcdef888888`
  - 难度：16^n

- **contains（包含）**：地址包含指定字符
  - 例如：`888888` 会匹配 `0x...abc888888def...`
  - 难度：相对较低

## 难度参考

| 字符长度 | 预估尝试次数 | 预计耗时（8核CPU） |
|---------|-------------|-------------------|
| 4位     | ~65K        | < 1秒             |
| 5位     | ~1M         | ~5-10秒           |
| 6位     | ~16M        | ~1-3分钟          |
| 7位     | ~268M       | ~20-40分钟        |
| 8位     | ~4.3B       | ~6-12小时         |

*注：实际速度取决于CPU性能，上述时间基于每秒10万次的速度估算*

## 性能优化建议

1. **使用高性能CPU**：CPU核心数越多，速度越快
2. **进程数设置**：建议设置为CPU核心数
3. **云服务器**：可以使用AWS、阿里云等高配置服务器
4. **长时间运行**：对于超级靓号（8位以上），建议使用screen或tmux在后台运行

## 安全提示

⚠️ **重要**：

1. 私钥是钱包的唯一凭证，必须妥善保管
2. 不要将私钥分享给任何人
3. 建议将私钥保存在安全的离线设备上
4. 生成的钱包文件（vanity_wallets.txt）包含私钥，请妥善保存

生成以"888888"开头的地址：

```
匹配模式: prefix
匹配字符: 888888

结果：
✓ 找到靓号！
  地址: 0x888888a1b2c3d4e5f6a7b8c9d0e1f2a3b4c5d6e7
  私钥: 0x1234567890abcdef...
  耗时: 45.32 秒
```

## 技术原理

1. 生成随机32字节私钥
2. 使用椭圆曲线算法（secp256k1）生成公钥
3. 对公钥进行Keccak-256哈希
4. 取哈希值的后20字节作为地址
5. 检查地址是否符合靓号要求
6. 如不符合，重复上述步骤

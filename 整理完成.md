## 🚀 现在如何使用？

### 方案1：本地使用
```bash
git clone https://github.com/six8888-cpu/bsclianghao.git
cd bsclianghao
pip3 install -r requirements.txt
./start.sh
```

### 方案2：服务器部署（推荐）
```bash
curl -o server_deploy.sh https://raw.githubusercontent.com/six8888-cpu/bsclianghao/main/server_deploy.sh
chmod +x server_deploy.sh
./server_deploy.sh
cd bsclianghao
python3 ultra_generator_v2.py
```

### 方案3：Windows
1. 访问：https://github.com/six8888-cpu/bsclianghao/actions
2. 下载最新的 `BSC-Vanity-Generator-Windows-x64.zip`
3. 解压并运行 `BSC-Vanity-Generator.exe`
